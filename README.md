## 🚀 Como executar
### `Backend`:

```bash
$ git clone https://github.com/rcapeto/kpto-social.git
$ cd backend
```

Para iniciar siga as instruções:
```bash
# Install dependencies
$ yarn
# Start project
$ yarn dev
```
O `servidor` vai ficar disponível no [endereço](http://localhost:3333).
